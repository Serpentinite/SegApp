# SegApp

## 概要
ポートフォリオ用のwebアプリケーションです。

## 説明
Instance SegmentationのMaskRCNNモデルを簡単に誰でも試すことができるアプリケーションです。<br>
画像をアップロードし、数十秒待つとマスクと分類したラベル名が表示されます。
